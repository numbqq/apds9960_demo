# APDS9960 Demo
### Getting started
```
$ sudo apt-get update
$ sudo apt-get install -y python-pip python-smbus
$ pip install apds9960
```

### Proximity sensor
```
$ sudo python test_prox.py
```

### Gesture sensor
```
$ sudo python test_gesture.py
```

### Color sensor
```
$ sudo python test_color.py
```
